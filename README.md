# Crag

Crag is a Flutter iOS field-board prototype for climbing sector notes, route intent, approach cues, and weather-window reads.

## Project Shape

- `lib/application` keeps the app shell and route entry.
- `lib/foundation` holds shared theme and formatting helpers.
- `lib/field_notes/domain` defines climbing-specific records and value objects.
- `lib/field_notes/data` keeps seeded notebook material.
- `lib/field_notes/presentation` builds the first usable overview screen.

The generated counter sample has been replaced with a sector notebook experience themed around outdoor crags.
